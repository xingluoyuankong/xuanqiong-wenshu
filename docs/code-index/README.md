# Code Index 说明

`docs/code-index/` 是“功能分区 + 索引”的固定目录，用来减少重复检索成本。

## 文件职责

- `functional-zones.md`
  - 人工维护的功能分区定义与核心文件入口（高信号导航）。
- `auto-file-index.md`
  - 自动生成的全项目文件索引（按分区输出，适合快速定位）。
- `auto-file-index.json`
  - 机器可读索引，适合脚本分析、自动化检查和后续工具接入。

## 分区原则

1. 先按“职责边界”分区：API、服务、领域模型、前端 UI、前端数据层、部署运维。
2. 再按“变更耦合”补充：经常一起改动的文件放在同一分区关注。
3. 避免跨区散射：任何优化任务先锁定一个主分区，再扩展到相邻分区。

## 更新方式

```powershell
python tools/generate_code_index.py
```

每次目录结构变化、模块重命名、功能迁移后都应该重新生成索引。
